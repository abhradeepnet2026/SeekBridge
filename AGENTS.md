# Agent Instructions

## Core Directives
- **DeepSeek First:** Delegate all possible tasks to DeepSeek models.
- **Gemma 4 Usage:** Avoid using Gemma 4 for task execution; use it only as a manager/dispatcher if necessary.
- **API Minimization:** Use provider APIs as infrequently as possible to reduce cost and token overhead.

## Ponytail Policy
- **Token Dieting:** Ponytail token consumption limitations apply ONLY to direct API models.
- **Bridged Models:** For models accessed via bridged solutions (e.g., SeekBridge), prioritize the most accurate and complete result over surgical brevity.
